# Uploading Files for Transcription

## Quick Answer
Drag and drop audio or video files onto the Recordings page, or click the Upload button to browse. Supported formats include MP3, WAV, M4A, MP4, MOV, and many more.

## Step-by-Step

1. Navigate to **Recordings** in the sidebar
2. Drag files onto the page, or click **Upload**
3. Enter a title for each recording
4. Optionally add metadata:
   - Date recorded
   - Participants
   - Description
   - Tags
5. Click **Save** or enable auto-transcribe

## Supported Formats

### Audio
- MP3, WAV, M4A, FLAC, OGG, AAC, WMA

### Video
- MP4, MOV, MKV, WEBM, AVI
- Video files are automatically converted to audio for transcription

## UI Location
- **Page**: Recordings
- **Key buttons**: Upload, drag-drop zone
- **Metadata panel**: Appears after upload

## Tips
- Upload multiple files at once for batch processing
- Use descriptive titles to find recordings later
- Add tags for easy filtering
- Large files (over 1GB) may take longer to process
- If upload fails, check available disk space
