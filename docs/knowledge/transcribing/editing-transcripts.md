# Editing Transcripts

## Quick Answer
Click any segment in the transcript to edit its text. You can also change speakers, add highlights for important parts, and leave comments on segments.

## Step-by-Step

### Edit Text
1. Open a recording's transcript view
2. Click on any segment's text
3. Edit the text directly
4. Press Enter or click outside to save

### Change Speaker
1. Click the speaker label (e.g., "SPEAKER_00")
2. Select a different speaker or create a new one
3. In the speakers panel, rename speakers to real names

### Add Highlight
1. Click the highlight icon on a segment
2. Choose a color: yellow, green, blue, red, purple, or orange
3. Click again to remove or change color

### Add Comment
1. Click the comment icon on a segment
2. Type your note
3. Click Save

## UI Location
- **Page**: Recording detail / Transcript view
- **Segments**: Listed with timestamps and speaker labels
- **Toolbar**: Above transcript with playback controls

## Tips
- Use keyboard shortcuts (J, K, L) to control playback while editing
- Highlights are great for marking key quotes or action items
- Comments are visible only in the app, not in exports
- The waveform above the transcript shows audio timeline
