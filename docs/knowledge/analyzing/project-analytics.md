# Project Analytics

## Quick Answer
View analytics for any project to see statistics, word frequency, recording timelines, and inherited tags. Access analytics from a project's detail page.

## Step-by-Step

1. Navigate to **Projects** in the sidebar
2. Click a project to open it
3. Click **Analytics** in the project header
4. View available insights

## Analytics Available

### Overview Stats
- Total recordings in project
- Total transcription duration
- Word count across all transcripts
- Average recording length

### Word Frequency
- Most common words and phrases
- Excludes common stop words
- Useful for identifying key themes

### Timeline
- When recordings were created
- Visual distribution over time
- Identify patterns in your data

### Tag Distribution
- Tags applied to recordings in the project
- Color-coded visualization
- Shows inherited vs direct tags

## UI Location
- **Page**: Projects > [Project Name] > Analytics
- **Access**: Analytics button in project header

## Tips
- Use word frequency to identify key topics
- Timeline helps spot recording patterns
- Compare analytics across projects
- Analytics update as you add recordings
