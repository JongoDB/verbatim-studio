# Using the AI Chat Assistant

## Quick Answer
Click the chat bubble in the bottom-right corner to open the AI assistant. Attach recordings or documents for context, then ask questions about your content.

## Step-by-Step

### Start a Chat
1. Click the **chat bubble icon** (bottom-right corner)
2. The chat panel opens

### Attach Context
1. Click the **attachment icon** (paperclip)
2. Choose what to attach:
   - **Recordings**: Includes their full transcripts
   - **Documents**: Includes extracted text
   - **Files**: Upload text files directly
3. Selected items appear as chips in the chat

### Ask Questions
1. Type your question in the input field
2. Press Enter to send
3. The AI responds using your attached context

### Save Conversations
1. Click the **save icon** in the chat header
2. Enter a title for the conversation
3. Find saved chats in the **Chats** page

## UI Location
- **Chat button**: Bottom-right corner (floating action button)
- **Chat panel**: Opens as overlay
- **Saved chats**: Chats page in sidebar

## Example Questions
- "Summarize the key points from this interview"
- "What action items were mentioned?"
- "Who talked about the budget and what did they say?"
- "Compare the two interviews I attached"
- "Extract all questions that were asked"

## Tips
- Attach multiple recordings for comparative analysis
- The AI only sees what you attach - be specific
- Clear the chat to start a new conversation
- Temperature setting affects creativity vs precision
- Saved conversations can be resumed later
