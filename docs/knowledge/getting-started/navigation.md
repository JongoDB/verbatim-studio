# Navigating Verbatim Studio

## Quick Answer
Use the sidebar on the left to navigate between pages. The main areas are Recordings (for transcription), Projects (for organization), and Settings (for configuration).

## Sidebar Items

### Main Navigation
| Item | What It's For |
|------|---------------|
| **Dashboard** | Overview of your library with stats and recent items |
| **Recordings** | Upload and manage audio/video files, start transcriptions |
| **Projects** | Organize recordings into groups with shared metadata |
| **Documents** | Upload and manage PDFs, images, and other documents |
| **Chats** | View and resume saved AI conversations |
| **Live** | Real-time transcription from your microphone (BETA) |
| **Search** | Find content across all recordings and documents |
| **Files** | Browse your file structure and move items |

### Bottom Section
| Item | What It's For |
|------|---------------|
| **Settings** | Configure transcription, AI models, storage, and more |
| **Theme toggle** | Switch between light, dark, and system themes |
| **Status indicator** | Green = connected, Yellow = connecting |
| **Version** | Current app version number |

## UI Location
- **Sidebar**: Always visible on the left (collapsible on desktop)
- **Mobile**: Tap the hamburger menu (three lines) to open sidebar

## Tips
- Click the collapse button (double chevron) to minimize the sidebar
- On mobile, tap outside the sidebar to close it
- The chat assistant (bubble icon) is always available in the bottom-right corner
