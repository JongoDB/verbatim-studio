# Using Tags

## Quick Answer
Tags are labels you apply to recordings for easy filtering. Add tags when uploading or editing recordings, then filter by tag on the Recordings page.

## Step-by-Step

### Add Tags to a Recording
1. Open a recording's detail view
2. Find the tags section
3. Click to add a tag
4. Type a new tag name or select existing
5. Tags are saved automatically

### Filter by Tag
1. Navigate to **Recordings**
2. Find the filter panel
3. Select one or more tags
4. Only matching recordings are shown

### Bulk Tag Recordings
1. Navigate to **Recordings**
2. Select multiple recordings (checkboxes)
3. Click the bulk actions menu
4. Select **Add Tags**
5. Choose tags to apply

## UI Location
- **Page**: Recordings (list and detail views)
- **Filter panel**: Left side or top of Recordings page
- **Tag display**: On recording cards and detail view

## Tag Best Practices

### Good Tag Examples
- Status: `completed`, `needs-review`, `archived`
- Priority: `high-priority`, `low-priority`
- Type: `interview`, `meeting`, `lecture`
- Year: `2026`, `2025`

### Tips
- Keep tag names short and consistent
- Use lowercase with hyphens for readability
- Tags can have colors for visual distinction
- Tags are inherited by projects containing the recording
