# Managing Documents

## Quick Answer
Upload PDFs and other documents to the Documents page. Verbatim Studio extracts text using OCR so you can search and include documents in AI chat context.

## Step-by-Step

### Upload a Document
1. Navigate to **Documents** in the sidebar
2. Click **Upload** or drag files onto the page
3. Wait for OCR processing to complete
4. The document is now searchable

### View a Document
1. Click a document card
2. The document viewer opens with:
   - Document preview
   - Extracted text
   - Notes panel

### Add Notes
1. Open a document
2. Click **Add Note**
3. Optionally anchor the note to a specific page
4. Type your note and save

## UI Location
- **Page**: Documents
- **Key buttons**: Upload
- **Status indicator**: Shows OCR processing status

## Supported Formats
- PDF (text and scanned)
- Images (PNG, JPG, TIFF) - OCR extracts text
- Word documents (DOCX)

## Tips
- OCR models must be downloaded first (Settings > OCR)
- Scanned PDFs take longer to process than text PDFs
- Include documents in AI chat for analysis
- Documents can be assigned to projects
- Search finds text within documents
