# Using the File Browser

## Quick Answer
The Files page shows your recordings and documents in a folder structure. Navigate folders, move files, and organize your library visually.

## Step-by-Step

### Navigate Folders
1. Navigate to **Files** in the sidebar
2. Click folders to open them
3. Use breadcrumbs at the top to go back
4. Click a file to open it

### Move Files
1. Select files (checkboxes)
2. Click **Move** or drag to a folder
3. Choose destination folder
4. Confirm the move

### Create Folders
1. Click **New Folder**
2. Enter a name
3. The folder appears in the current location

## UI Location
- **Page**: Files (labeled "Files" in sidebar)
- **Breadcrumbs**: Top of page, shows current path
- **View options**: List or grid view

## Tips
- The file browser reflects your storage location structure
- Recordings and documents appear as files
- Folder organization is separate from project organization
- Sort by name, date, size, or type
