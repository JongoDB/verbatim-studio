# Organizing with Projects

## Quick Answer
Create projects to group related recordings together. Navigate to Projects, click New Project, and then assign recordings to it from the Recordings page.

## Step-by-Step

### Create a Project
1. Navigate to **Projects** in the sidebar
2. Click **New Project**
3. Enter a name and optional description
4. Select a project type (optional, for custom metadata)
5. Add tags for filtering
6. Click **Create**

### Assign Recordings to a Project
1. Navigate to **Recordings**
2. Select one or more recordings (checkbox)
3. Click the bulk actions menu
4. Select **Add to Project**
5. Choose the target project

### View Project Contents
1. Navigate to **Projects**
2. Click a project card
3. View all recordings in that project
4. Access project analytics

## UI Location
- **Page**: Projects
- **Key buttons**: New Project, Add to Project (in Recordings)
- **Project card**: Shows name, recording count, tags

## Project Types
Project types define custom metadata fields. For example, a "Research Interview" type might include fields for:
- Interviewer name
- Interviewee name
- Interview date
- Research topic

Create custom project types in Settings or when creating a project.

## Tips
- Use projects for logical groupings (by client, event, research topic)
- Use tags for cross-cutting categories (priority, status, year)
- One recording can belong to multiple projects
- View project analytics for word frequency and timelines
